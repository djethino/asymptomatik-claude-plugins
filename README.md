# ASymptOmatik Claude Plugins

Official plugin marketplace for Claude Code by ASymptOmatik.

## Installation

```bash
/plugin marketplace add djethino/asymptomatik-claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [claude-guardian-coach](https://github.com/djethino/claude-guardian-coach) | Behavioral guidance and context preservation |

## Install a Plugin

```bash
/plugin install claude-guardian-coach
```
